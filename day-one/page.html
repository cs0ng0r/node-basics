What is Node.js?

Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine. It allows you to run JavaScript on the server-side, which can be used for building scalable network applications.

Exercise 1: Setting up Node.js

Ensure you have Node.js installed on your system. You can check by running the following command in your terminal: 

node -v

If you don't have it installed, download it from Node.js official site.

Your First Node.js Script

Let's create a simple "Hello, World!" program in Node.js. Open a text editor, and write the following: 

console.log('Hello, World!');

Save the file as hello.js and run it from the terminal: 

node hello.js

If done correctly, it will output Hello, World!.

Working with Node.js Modules

Node.js has a built-in module system. To include a module in your project, use the require function. For example, the fs module allows you to work with the file system: 

const fs = require('fs');
fs.writeFileSync('message.txt', 'Hello, Node.js!');

This will create a file named message.txt with the text "Hello, Node.js!"

Creating a Simple Web Server

Node.js allows you to create web servers easily using the http module. Here's an example of a simple server: 

const http = require('http');
const server = http.createServer((req, res) => {
    res.statusCode = 200;
    res.setHeader('Content-Type', 'text/plain');
    res.end('Hello, this is your server speaking!');
});

server.listen(3000, () => {
    console.log('Server running at http://localhost:3000/');
});

Run this script and open http://localhost:3000 in your browser. You should see the message: "Hello, this is your server speaking!"
